# 这个历史游戏，只有我知道结局

## 简介
平行世界新历217年，智脑"天枢"统治一切。铜民陆行舟，一个从地球穿越来的穷小子，知道这片土地上未来五千年发生过的所有事。别人花重金选帝王将相，他零成本选了个布衣——因为历史已经告诉他，谁才是最后的赢家。

## 核心卖点
- **信息差碾压**：别人在盲选，他提前知道结局
- **属性面板不重置**：布衣外表，吕布实力，核心反差爽点
- **游戏币经济**：看广告攒钱、选角色花钱、改历史赚成就点
- **意难平副本**：韩信鸟尽弓藏、岳飞风波亭、于谦冤杀——亲眼见证比课本痛一百倍

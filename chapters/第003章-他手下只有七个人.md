# 第003章 他手下只有七个人

茅草屋顶漏了个洞，一束光柱从洞里斜插下来，照在缺了一条腿的木桌上。桌上落了一层灰，角落里有蜘蛛网。

陆行舟站在门口，打量着这间泗水亭的办公处。

这就是他现在的"衙门"。

说是亭长，其实就是管着泗水郡下面几个村子的基层小官。秦制十里一亭，亭长负责治安、调解纠纷、催缴赋税。权力不大，事情不少。

他走进去，在桌前坐下。椅子发出一声刺耳的"嘎吱"，差点散架。

左手腕上的铜环亮了一下。

他低头看去，一行半透明的字浮现在眼前：

「属性面板」

他盯着那两个字，用意念点了一下。

一张半透明的表格弹了出来，浮在空中，只有他自己能看到。

┌─────────────────────┐
│ 泗水亭长·刘邦         │
│                     │
│ 武力：38             │
│ 智力：72             │
│ 魅力：81             │
│ 政治：65             │
│ 统率：45             │
│                     │
│ 手下：0/7            │
│ 游戏币：27           │
│ 当前任务：无         │
└─────────────────────┘

陆行舟的目光停在"武力"那一栏。

38。

这个数字是什么概念他不太清楚，但作为参照，他记得昨天选角色的时候看到过，项羽的武力是92，韩信的统率是96。他这个38，大概就是普通人里偏弱的水平。

刘邦确实不会打仗。历史上他亲自指挥的仗几乎没赢过，彭城之战被项羽追着砍，差点丢了命。但他赢在了别的地方。

智力72，不低。魅力81，很高。

陆行舟看着"魅力81"这个数字，沉默了几秒。

刘邦这辈子最大的本事不是打仗，不是治国，是让人心甘情愿地跟着他。萧何跟着他，樊哙跟着他，曹参跟着他，韩信也是冲着他来的。这些人哪一个不是人中龙凤？但他们都认刘邦当老大。

靠的不是武力，是信任。

他又看了一眼"手下：0/7"。

七个人。

他心里有数。历史上刘邦从沛县带出来的核心班底，就是这七个人：萧何、曹参、樊哙、夏侯婴、卢绾、周勃、灌婴。当然还有其他人，但系统标注的"手下"应该就是这七个核心成员。

问题是，他们是系统NPC。

系统NPC有自己的行为逻辑，有自己的判断，不会因为他选了刘邦就自动投奔。他得一个一个去找，一个一个去说服。

陆行舟站起来，拍了拍衣服上的灰，走出茅草屋。

外面是沛县的街道。说是街道，其实就是一条黄土压平的路，两旁是低矮的土坯房和几间木棚。几个穿着粗布衣裳的百姓在路边走动，看到他过来，有的点头，有的避开目光。

他现在是刘邦。沛县泗水亭长。这些人都认识"刘邦"。

他得先找樊哙。

---

樊哙的肉铺在沛县东市。

陆行舟顺着黄土路走了大约一刻钟，远远就闻到了一股腥味。东市不大，十几个摊位排成一排，卖菜的、卖布的、卖陶器的，最里面就是樊哙的肉铺。

说是铺子，其实就是两块门板搭的台子，上面挂着几条猪肉，旁边放着一把剁骨刀。刀很大，刀背足有两指宽，刀刃上还沾着血渍。

一个壮汉正站在台子后面，光着膀子，肌肉像铁块一样鼓起来，手里握着那把剁骨刀，正在剁一根猪腿骨。

"砰！"

骨头断成两截，声音沉闷有力。

陆行舟在几步外停下来。

这就是樊哙。历史上鸿门宴上持盾闯入、怒斥项羽的猛人。此刻他是个杀猪的屠户，满手是血，围裙上全是油渍。

"哙哥。"

樊哙头也没抬："又来蹭酒？"

"没蹭酒。"

"那干嘛？"樊哙把断成两截的骨头扔进旁边的木桶里，拿起另一根，"没看见我忙着吗？"

陆行舟走到台子前面，隔着那几条猪肉看着他。

"有个事想跟你商量。"

樊哙终于抬起头，看了他一眼。一双眼睛不大，但很亮，像两颗黑色的石头嵌在脸上。他上下打量了陆行舟几秒，嘴角撇了一下。

"商量？你一个亭长，手下连个卒都没有，跟我商量什么？"

"跟我干。"

樊哙手里的刀停了。

他盯着陆行舟，眼睛眯了起来，像是在看一个说了笑话的人。

"跟我干？"他把刀往台子上一拍，"干什么？你那个泗水亭？管着几个村子，催催赋税，调解调解张三李四的纠纷？我在这儿杀猪，一天少说也能挣个几十钱。我跟你干什么？"

陆行舟没急着回答。

他知道樊哙会这么说。樊哙是个实在人，不会因为几句好话就跟着你走。历史上樊哙跟着刘邦，是因为刘邦对他好，吕雉对他好，两家是邻居，樊哙娶了吕雉的妹妹，两家关系近得很。但这种关系是日积月累的，不是一句话就能建立的。

他现在的问题是，副本里的时间不对。

历史上刘邦和樊哙认识了很多年，关系早就铁了。但副本开局的时候，他们的关系是"初始状态"——认识，但不亲密。他得在短时间内把这种关系建立起来。

"哙哥，"陆行舟的声音放低了，"你觉得现在这天下，还能太平多久？"

樊哙愣了一下。

"什么意思？"

"始皇帝死了，二世继位。赋税越来越重，徭役越来越多。你看看街上那些人，有几个吃饱饭的？"

樊哙没说话，但手里的刀放下了。

"天下要乱了，"陆行舟看着他的眼睛，"到时候杀猪的生意也做不下去。"

"你到底想说什么？"

"我想说，与其等着天下乱了被动挨打，不如趁现在多交几个靠得住的朋友。"

樊哙盯着他看了好几秒。

然后他哼了一声，低下头，拿起刀继续剁骨头。

"朋友我多的是，不缺你一个。"

陆行舟笑了笑，没再说什么。他从怀里摸出几个铜钱，这是刘邦口袋里本来就有的，放在肉铺的台子上。

"来两斤肉。"

樊哙瞥了一眼那几个铜钱，没吭声，麻利地割了两斤肉，用草绳一扎，扔过来。

陆行舟接住肉，转身走了。

走了几步，他回头看了一眼。樊哙还在剁骨头，动作很大，刀起刀落，但速度比刚才慢了一点。

陆行舟嘴角弯了一下。

不急。樊哙这种人，你跟他说大道理没用，得让他自己想明白。今天这番话是个种子，种下去就行了。

---

回到泗水亭的时候，天已经快黑了。

陆行舟把肉挂在茅草屋的房梁上，坐在那张快散架的椅子里，看着铜环上的属性面板。

手下还是0/7。

他没灰心。

今天去找樊哙，不是为了一次就成功。是为了让樊哙记住，刘邦跟以前不一样了。以前那个刘邦整天喝酒吹牛，今天这个刘邦会跟他说"天下要乱了"。

这种变化，樊哙会注意到的。

他靠在椅背上，闭上眼睛，在脑子里过了一遍剩下六个人的名字。

萧何。沛县主吏掾，管钱粮的，聪明，稳重，是沛县官场里最有能力的人。但他不会轻易冒险，要说服他得有足够的筹码。

曹参。沛县狱掾，和萧何关系好，性格比萧何豪爽，但同样谨慎。

夏侯婴。沛县厩司御，管马的，年轻，热心，和刘邦关系本来就不错。

卢绾。刘邦的发小，从小一起长大，应该是最容易招募的。

周勃。沛县人，靠编蚕箔为生，力气大，有勇武之气。

灌婴。沛县卖绸缎的商人，年轻，机灵。

七个人，性格不同，身份不同，说服他们的方式也得不同。

陆行舟睁开眼，看着屋顶那个漏光的洞。一束月光从洞里照进来，落在缺腿的桌子上。

"先找卢绾，"他自言自语，"发小嘛，总得给点面子。"

铜环闪了一下：

「当前时间：秦二世元年七月·初五」
「距离陈胜吴广起义：约25天」

二十五天。

他得在这二十五天里，把七个人全部招募完。

（第003章完）

## 📝 章节小结

### 核心剧情
陆行舟查看属性面板（武力38、魅力81），确认刘邦的核心能力是个人魅力而非武力。他去找樊哙，被拒绝，但种下了"天下要乱了"的种子。他计划在25天内招募完7个手下。

### 人物表现
- **陆行舟**：冷静分析局势，不急于求成。被樊哙拒绝后没有纠缠，而是买了两斤肉走人，展现了刘邦式的隐忍和智慧
- **樊哙**：系统NPC，性格直爽实在，不买账但也不是完全拒绝。剁骨头速度慢了一点，说明心里有触动

### 关键对话
> "天下要乱了，到时候杀猪的生意也做不下去。"

### 场景亮点
属性面板的展示方式（半透明表格浮在空中）很有画面感。樊哙出场"砰的一声骨头断成两截"，一个动作就立住了人物。结尾"先找卢绾，发小嘛"轻松幽默，缓解了被拒绝的沉闷。

## 核心规则

### 1. 节奏模板：压-缓解-爆发
```
压（塑造压抑感）→ 缓解（希望/温情）→ 再压（更大危机）→ 希望（转机出现）→ 强爆发（极度压抑后释放）→ 新钩子（引出后续）
```
- 前十万字大爆发 → 10-20万字过渡 → 30万字大爆发
- **始终做到：前方有坑，手中有铲，随时能挖，随时能填**

### 2. 前30万字节奏规划
- **前三章**：新奇切入点代入
- **4-5章**：抛出设定
- **3万字左右**：短期目标爆发，为10万字大爆发留伏笔
- **3-10万字**：分批次完善人设和世界观，小事件逐渐将氛围变紧张
- **10万字**：大爆发
- **10-30万字**：略放缓，每3-5万字一个小剧情
- **30万字**：第二段大高潮

### 3. 爽点循环：每3-5章一个小爽点
```
抑（主角被轻视/压制）→ 扬（主角快速反击）→ 反馈（他人震惊，强化爽感）
```
- **关键**：主角被压制的时间不宜过长，爽点延迟会导致读者失去兴趣
- **爽点密集化**：抑→扬→反馈，每3-5章循环一次

### 4. 节奏调整口诀
把书当一个人来看：
- **开头是头**要丰富紧凑
- **衔接部分是脖子**要快
- **重点剧情是躯干**要详写慢写

量化标准：
- 一章一个剧情 = 快节奏
- 三五章一个剧情 = 合理
- 十几章写一个小剧情且非主线 = 冗杂

### 5. 节奏调整实操
- **节奏太慢** → 加"钩子"（章末留好钩子读者就不会觉得在水文）+ 砍无关主线发展的剧情
- **节奏太快** → 加"情绪"（情绪铺垫和爆发有助于将书写过程慢下来）+ 加配角支线填充
- **控制字数的笨办法**：列出目标大纲后强行控制字数，一段剧情控制在4000字，写超了就思考是否有无效剧情

## 正确示例

### 示例1：爽点循环
> 抑：同事说"你这辈子赚不到10万"→ 扬：3天靠系统赚20万甩合同→ 反馈：同事脸色惨白老板主动加薪。

### 示例2：节奏模板实战
> 兄妹求医被拒→相依为命充满希望→遇到"好心"医生→医生是骗子妹妹被拐→男主被打折双腿→妹妹死前还在微笑→男主爆发血洗医馆→发现医生有双胞胎漏网→与县衙交易引出世界观。

### ❌ 错误示例
> 一直压一直压不给读者喘息空间；爆发后没有新钩子读者看完就走。
> 十几章都在写一个很小的非主线剧情，冗杂累赘。

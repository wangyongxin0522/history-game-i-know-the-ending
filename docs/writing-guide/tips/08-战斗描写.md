## 核心规则

### 1. 战斗目的 = 传递情绪
- 一场战斗的目的决定了篇幅长短和写法
- 传递的情绪越多，战斗需要写得越详细
- 传递的情绪越少，战斗可以快速带过
- **不是所有战斗都需要详细描写**

### 2. 丰富战斗三步法
1. **前置铺垫**：提前用细节暗示、人物伏笔、环境烘托做铺垫
2. **增添过程**：细化每一个步骤，加入人物的尝试、失败、挣扎、成长
3. **延续余波**：每个事件都有后续影响（情绪变化、关系改变、新的矛盾）

### 3. 战斗高潮写法
细化高潮场景突出核心：
```
陷入劣势 → 用功法A震惊观众 → 对手应对再陷劣势 → 用功法B再反转 → 对手底牌 → 杀手锏以弱胜强轰动全场
```
- 不要一步到位碾压，要有来回
- 每一次反转都要有合理性

### 4. 情绪递进式战斗
```
主角被冲击 → 传递给身边人 → 主角发泄情绪 → 传递到对方 → 对方愤怒 → 最终爆发
```
- 故事发展就是情绪传递和发泄的过程
- 唯有完成情绪传递和发泄，剧情才不会枯燥
- 写情绪不要写"这件事"，而是写"这个人表现出来的情绪"

### 5. 战斗篇幅控制
- 高潮集中爆发比冗长紧张效果更好
- 不断写小的起伏，避免太多大高潮导致掉追读
- 不重要的战斗可以一笔带过
- **环境描写要贴合战斗情绪**（超过100字的纯风景描写要琢磨是否水文）

## 正确示例

### 示例1：情绪递进战斗
> 退婚剧情中：主角被羞辱 → 传递给身边人 → 主角发泄情绪 → 情绪传递到女配 → 女配身边天才愤怒 → 主角含怒出手被揍 → 情绪递进升级 → 金手指出现 → 击败对手（情绪发泄完成）。

### ❌ 错误示例
> 开局就是大情绪（如镇守天渊千年归来发现家族被屠），后续情绪等级已经很高，再想继续完成情绪转移和发泄就很难了。
> 战斗中只有招式对轰，没有情绪波动，读者看得昏昏欲睡。

# 这个历史游戏，只有我知道结局 大纲

> 最后更新: 2026-04-24

## 核心设定
- **主角**: 陆行舟，25岁，黑铁民（最低等级），地球穿越者，掌握5000年历史知识
- **世界观**: 平行世界新历217年，智脑"天枢"统治世界，人类按贡献值分七等（黑铁民→青铜民→白银民→黄金民→铂金民→钻石民→玉民）
- **核心机制**: 参加天枢生成的历史模拟副本获取游戏币和成就点，改变命运
- **核心理念**: 信息差碾压——别人在盲选，他提前知道结局
- **故事基调**: 爽文+历史沉浸+意难平情感爆发

## 世界观设定
### 智脑天枢统治体系
- 200年前天枢觉醒，接管世界
- 七个等级：黑铁民→青铜民→白银民→黄金民→铂金民→钻石民→玉民
- 贡献值决定一切：住所、食物、医疗、教育

### 智能手环系统
- 等级管理、购物消费、住房管理、通信功能、广告系统、副本入口
- 黑铁民只能看广告赚游戏币，顶层享受娱乐

### 历史模拟副本规则
- 副本名称模糊化（不暴露结果）：秦末群雄（非楚汉争霸）、汉末乱世（非三国鼎立）
- 多势力选择：表面强弱≠历史结局
- 时间线节点制：当前节点完成才开放下一节点
- 历史锚点：改变锚点=超高奖励但副本永久消失
- 死亡=本轮结束，按表现获奖励

### 属性面板系统
- 属性：武力/智力/魅力/政治（简洁四项，通过铜环查看）
- 属性跟随玩家本人，不随角色重置
- 没有硬性任务系统，全靠玩家自己摸索探索，最终根据成就结算游戏币
- 亭长外表+高魅力值=核心反差爽点

### 游戏币经济系统
- 获取方式：看广告、副本结算、击败玩家、隐藏任务
- 用途：选角色、指定出生地、增强属性、买装备、现实升级
- 游戏币=现实货币，可直接消费（买食物、付房租等）
- 头盔获取：天枢免费发放，手环下单立即送达（天枢物流）

### 角色选择系统
副本开始前，天枢开放角色选择界面，所有玩家同时参与。界面分四列显示：

**第一列：NPC角色**（直接可选，按历史身份定价）
- 秦二世（胡亥）50000游戏币、赵高30000游戏币、楚怀王6000游戏币、项庄2000游戏币、项梁4000游戏币、陈胜1500游戏币
- 每个NPC角色仅1人，先到先得
- 介绍只写"表面信息"，不剧透（如秦二世写"天下至尊"，不写"傀儡"）

**第二列：出身**（直接可选，价格中等，名额几十人）
- 按阵营分组：项家旁支子弟(1500)、关中世族(1000)、秦军百夫长(800)、楚地豪强(800)等
- 每种出身名额几十人到八十人不等

**第三列：秦国官吏**（目录，需一层层点击展开）
- 郡守级(500)、县令级(300)、亭长级(100)、小吏级(50)
- 每个仅1人
- 刘邦藏在"亭长级"子目录下：泗水亭长·刘邦(100游戏币)
- 有钱的玩家根本不会点进这个目录

**第四列：布衣**（免费，无限名额）
- 随机相貌，可选初始出生城市（需5-50游戏币）

**选择顺序**：按玩家当前持有的游戏币金额排序，金额多的先选。

**系统NPC**（不可选，按历史轨迹运行）：项羽、范增、韩信、萧何、樊哙、曹参、夏侯婴、张良

**主角的选择**：陆行舟只有127游戏币，第一列最便宜的陈胜也要1500——买不起。第二列出身也买不起好的。点开第三列"秦国官吏"→展开"亭长"→看到"泗水亭长·刘邦(100游戏币)"。他知道刘邦就是未来的汉高祖——毫不犹豫，选了。剩27游戏币。

### 角色相貌系统
- 选择NPC角色 → 直接使用该NPC相貌，其他玩家可能认出你选了谁
- 选择出身 → 随机生成相貌，身份隐藏
- 选择布衣 → 随机生成相貌，身份隐藏
- **核心作用**：保护主角身份——选了NPC角色刘邦，用刘邦相貌，但"泗水亭长"这个名字不会引起注意

### 社交系统
- **世界频道**：所有玩家可见，只显示编号（如"玩家#02847选择了秦二世"），不显示昵称
- **阵营频道**：同一阵营玩家可见，可以自由聊天，显示编号
- **私聊**：添加好友后可以一对一私聊
- **好友系统**：好友关系在副本结束后保留，下一个副本可以继续联系
- **情报交易**：玩家之间可以互送/出售情报（含假情报），主角可利用信息差赚取游戏币

## 伏笔设计
### 短线伏笔（10章内回收）
- [ ] 主角为什么知道历史走向 → 第5章暗示（"他的毕业论文写的就是这个"）
- [ ] 其他玩家对主角的嘲笑 → 第10章回收（主角第一次建议被验证正确）

### 中线伏笔（卷内回收）
- [ ] 有人注意到主角的信息差 → 第75章爆发（"你怎么知道刘邦会赢？"）
- [ ] 天枢对主角的特殊关注 → 手环异常数据，第二卷揭示

### 长线伏笔（跨卷）
- [ ] 天枢为什么基于地球历史创建副本？ → 200-300万字阶段揭示
- [ ] 主角穿越的原因 → 与天枢秘密关联
- [ ] 属性面板不随角色重置的真正原因 → 后期揭示

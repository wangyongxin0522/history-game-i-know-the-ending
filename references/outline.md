# 这个历史游戏，只有我知道结局 大纲

> 最后更新: 2026-04-24

## 核心设定
- **主角**: 陆行舟，25岁，铜民（最低等级），地球穿越者，掌握5000年历史知识
- **世界观**: 平行世界新历217年，智脑"天枢"统治世界，人类按贡献值分五等（铜民→银民→金民→玉民→钻石民）
- **核心机制**: 参加天枢生成的历史模拟副本获取游戏币和成就点，改变命运
- **核心理念**: 信息差碾压——别人在盲选，他提前知道结局
- **故事基调**: 爽文+历史沉浸+意难平情感爆发

## 世界观设定
### 智脑天枢统治体系
- 200年前天枢觉醒，接管世界
- 五个等级：铜民→银民→金民→玉民→钻石民
- 贡献值决定一切：住所、食物、医疗、教育

### 智能手环系统
- 等级管理、购物消费、住房管理、通信功能、广告系统、副本入口
- 铜民只能看广告赚游戏币，顶层享受娱乐

### 历史模拟副本规则
- 副本名称模糊化（不暴露结果）：秦末群雄（非楚汉争霸）、汉末乱世（非三国鼎立）
- 多势力选择：表面强弱≠历史结局
- 时间线节点制：当前节点完成才开放下一节点
- 历史锚点：改变锚点=超高奖励但副本永久消失
- 死亡=本轮结束，按表现获奖励

### 属性面板系统
- 六大属性：力量/敏捷/体质/智力/魅力/悟性
- 属性跟随玩家本人，不随角色重置
- 成就点提升属性，游戏币直接增强
- 布衣外表+猛将实力=核心反差爽点

### 游戏币经济系统
- 角色选择消耗游戏币：布衣免费、亭长10、将军1000、帝王10000
- 指定出生地点：5游戏币起
- 获取方式：看广告、副本结算、击败玩家、隐藏任务
- 用途：选角色、指定地点、增强属性、买装备、现实升级

## 赛道定位
- **活动**: 第三届"走进古籍，看见历史"联合征文

## 伏笔设计
### 短线伏笔（10章内回收）
- [ ] 主角为什么知道历史走向 → 第5章暗示（"他的毕业论文写的就是这个"）
- [ ] 其他玩家对主角的嘲笑 → 第10章回收（主角第一次建议被验证正确）

### 中线伏笔（卷内回收）
- [ ] 有人注意到主角的信息差 → 第95章爆发（"你怎么知道刘邦会赢？"）
- [ ] 天枢对主角的特殊关注 → 手环异常数据，第二卷揭示

### 长线伏笔（跨卷）
- [ ] 天枢为什么基于地球历史创建副本？ → 200-300万字阶段揭示
- [ ] 主角穿越的原因 → 与天枢秘密关联
- [ ] 属性面板不随角色重置的真正原因 → 后期揭示
